## How to start

1. 安装依赖 `requirements.txt`
2. 将准备好的资源包(res与data)解压在根目录
   ```
   Anisebot
   ┣ anise_bot
   ┣ anise_core
   ┣ config
   ┣ data
   ┣ res
   ┣ ...
   ```
3. run your bot using `nb run`
